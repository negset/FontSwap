Font Swap

What's this?

A Chrome extension that replaces the fonts specified in a website's stylesheet with the fonts of your choice.

For example, if you set it to replace “Segoe UI” with “Arial”, then “Arial” will be used on all sites where “Segoe UI” is specified.

You can change the font settings in the extension options.
